# 测试计划 - JPetStore 电商平台

## 1. 测试范围

### 功能模块
| 模块 | 说明 | 优先级 |
|------|------|--------|
| 用户注册 | 新用户注册、邮箱验证 | P0 |
| 用户登录 | 登录/登出、记住密码 | P0 |
| 商品浏览 | 分类浏览、搜索、分页 | P0 |
| 购物车 | 添加/删除商品、修改数量 | P0 |
| 下单结算 | 填写地址、选择支付、提交订单 | P0 |
| 个人中心 | 修改资料、查看订单 | P1 |

## 2. 测试策略

### 功能测试
- 等价类划分 + 边界值分析设计用例
- 每个功能点覆盖正常流程 + 异常流程

### 兼容性测试
- Chrome / Edge / Safari 最新版
- 移动端适配（375px-414px 宽度）

## 3. 测试用例编写规范

每条用例包含：
- ID、模块、用例标题、前置条件、测试步骤、预期结果、优先级
