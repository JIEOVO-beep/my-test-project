# Bug 报告示例

## Bug #001 - 登录成功但未跳转到首页

| 字段 | 内容 |
|------|------|
| **标题** | [登录] 输入正确账号密码后页面无跳转 |
| **环境** | Chrome 120.0 / Windows 11 |
| **版本** | JPetStore v1.0 |
| **严重程度** | 严重 |
| **优先级** | P0 |

### 前置条件
已注册账号 testuser / password123

### 重现步骤
1. 打开 https://petstore.octoperf.com/
2. 点击右上角 "Sign In"
3. 输入用户名: testuser
4. 输入密码: password123
5. 点击 "Login" 按钮

### 实际结果
- 登录按钮点击后页面刷新，URL 发生变化
- 但没有跳转到用户首页，仍然停留在登录页
- 查看网络请求，登录接口返回 200 状态码

### 预期结果
登录成功后自动跳转到网站首页，右上角显示用户名
